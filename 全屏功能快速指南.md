# 📱 图表全屏功能 - 快速使用指南

## 🎯 一键全屏横屏显示

### 快速开始

```
1️⃣ 找到全屏按钮
   图表右上角 → ⛶ 图标

2️⃣ 点击进入全屏
   图表占满整个屏幕 + 自动横屏

3️⃣ 退出全屏
   点击 🗗 按钮 或 按 ESC 键
```

## 核心功能

### ✅ 自动横屏
- 进入全屏后自动尝试横屏锁定
- 更适合查看数据趋势

### ✅ 响应式布局
- 图表高度：100vh（全屏高度）
- 自动适应不同设备

### ✅ 多种退出方式
- 点击退出按钮 🗗
- 按键盘 ESC 键
- 手势下滑（移动设备）

### ✅ 保留所有功能
- 切换图表类型（折线/柱状/饼图）
- 切换时间周期（日/周/月/年）
- 移动平均线开关

## 常见问题

### Q: 全屏按钮在哪里？
**A:** 图表右上角，"收起"按钮旁边的 ⛶ 图标

### Q: 如何退出全屏？
**A:** 三种方式：
1. 点击 🗗 按钮
2. 按 ESC 键
3. 从屏幕顶部下滑（移动端）

### Q: 为什么没有自动横屏？
**A:** 可能原因：
- 浏览器不支持方向锁定
- 设备屏幕旋转被禁用
- 手动旋转设备即可

### Q: iOS Safari 无法全屏？
**A:** 
- 需要 iOS 16.4+ 版本
- 或将应用添加到主屏幕

### Q: 全屏时可以切换图表类型吗？
**A:** 可以！所有控制功能都保留：
- 📈 折线图
- 📊 柱状图
- 🥧 饼图
- 日/周/月/年切换
- 移动平均线
